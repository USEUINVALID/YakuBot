Бот для круток с генша и аниме картиночек

``` За авторством Buddha и Darkness```

- Список доступных команд:
    
    - "дать крутки" - админская команда для выдачи круток

    - "получить крутки" - команда общего пользования для получения 5 круток раз в час

    - "крутка" - открутить 1 раз

    - "крутки" - открутить 10 раз
  
    - "инвентарь" - посмотреть свой инвентарь
    
    - "пикча + тег" - получить аниме пикчу/гифку

    - "пивасик" - картиночка эльфийки с банкой

    - "капибара" - гифка капибары

    - "никнейм" - изменить свой никнейм

``В будущем бот будет дорабатываться, ждите обновлений!``
